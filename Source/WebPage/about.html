Prescription drug claims analysis

This Usecase is to understand and predict prescription drug usage of medic-aid population, which will enable the local government and health plans to reduce spend. Combine datasets for analysis. Analyse and visualize interesting patterns in the data.

  1. Pre-Processing: Microsoft Excel
  2. Scripting Language: Shell Script, Curl
  3. Visualization: Tableau Desktop
  4. Data Categorzation: Java Program
  5. Web Page: HTML 5.0, Javascript, CSS

Where the Data is Collected From
